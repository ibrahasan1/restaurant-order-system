# 🖥️ Hardware-Empfehlungen

## Lokaler Server (Mini-PC)

### Budget-Option (~150-200€)
- **Raspberry Pi 5 (8GB)** oder **Beelink Mini S12 Pro**
- Ausreichend für 10-20 gleichzeitige Verbindungen
- Niedriger Stromverbrauch (ideal für Dauerbetrieb)

### Empfohlene Option (~300-400€)
- **Intel NUC** oder **Beelink SER5**
- 8GB RAM, 256GB SSD
- Zuverlässiger Dauerbetrieb

### Wichtig
- **SSD statt HDD** (schneller, leiser, zuverlässiger)
- **UPS / USV** (unterbrechungsfreie Stromversorgung, ~50€)
- **Ubuntu Server 24.04 LTS** als Betriebssystem (kostenlos)

## WLAN-Router

### Anforderungen
- **Dual-Band** (2.4 GHz + 5 GHz)
- Mindestens **Wi-Fi 5 (802.11ac)**
- Statische IP-Vergabe möglich

### Empfehlungen
- **TP-Link Archer AX21** (~60€) – Gutes Preis-Leistungs-Verhältnis
- **Fritz!Box 7590** (~200€) – Wenn ohnehin vorhanden

### Konfiguration
1. Statische IP für den Server vergeben (z.B. `192.168.1.100`)
2. 5 GHz Band für die Tablets bevorzugen (weniger Interferenz)
3. DHCP-Bereich einschränken
4. Separates WLAN für das Bestellsystem (optional, aber empfohlen)

## Kellner-Handys (3x)

### Budget (~100-150€/Stück)
- Jedes Android-Handy ab Android 10
- **Samsung Galaxy A15** oder **Xiaomi Redmi Note 13**
- Mindestens 4GB RAM

### Tipp
- Gebrauchte iPhones/Androids funktionieren auch
- Chrome oder Safari als Browser
- App als PWA zum Homescreen hinzufügen

## Tablets für Küche (3x) und Bar (2x)

### Budget (~150-200€/Stück)
- **Samsung Galaxy Tab A9** (10.9")
- **Lenovo Tab M10 Plus** (10.6")
- Mindestens 3GB RAM, 10"+ Display

### Tipps
- **Querformat-Halterung** montieren
- **Dauerstrom** per USB-Kabel (Akku schonen)
- **Display-Helligkeit** auf Maximum
- **Auto-Lock deaktivieren** (Display immer an)
- **Kiosk-Modus** einrichten (verhindert versehentliches App-Wechseln)
